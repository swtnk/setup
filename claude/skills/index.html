Use the installer

The install script creates CLAUDE.md and .claude/rules/ in your project. Claude Code reads these automatically. No manual setup needed.

$ curl -fsSL https://swetanksubham.com/setup/claude/vscode/install | bash
or
Claude.ai

Install the .skill file

Download the .skill file and install it in Claude.ai to get the same coding standards in the web/app interface.

  1. Download the .skill file below
  2. Open claude.ai
  3. Go to Settings (bottom-left) → Skills
  4. Click Add Skill and upload the file
