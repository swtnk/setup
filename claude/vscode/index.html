Claude Code Setup

One command to set up architecture-first coding standards for Claude Code in VS Code.

Install
--copilot Also set up GitHub Copilot instructions
--global Install to ~/.claude/ for all projects
--force Overwrite existing files
--no-detect Skip language detection, install all rules

Architecture First

Claude analyzes your entire codebase before writing a single line. No more context loss.

SOLID Always

Every generated file follows SOLID principles. No exceptions.

Immutable by Default

All variables const/final/readonly unless mutation is explicitly justified.

Zero Hardcoded Values

All constants in centralized config files. Clean, maintainable code.

Self-Explanatory Code

No comments needed. Descriptive names, clean structure, code that reads like prose.

Senior Engineer Level

Right abstractions, edge case coverage, precise naming. Passes principal-level review.

Task Decomposition

Big problems broken into small tasks with a plan. Tracked in .claude/context/.

Tests Built In

Unit and integration tests written alongside implementation, not as an afterthought.

Project Memory

Persistent .claude/context/ directory so Claude remembers across sessions.

Auto-Detection

Detects Python, Java, Rust, TypeScript, Go, C/C++ and installs matching rules.

Follows Up

Asks when ambiguous, presents trade-offs, checks in at milestones. Never assumes.

Matches Your Style

For existing codebases, mirrors every convention. Generated code is indistinguishable.

What it creates

your-project/
├── CLAUDE.md                  # Main instructions (Claude reads this)
├── .claude/
│   ├── rules/
│   │   ├── general.md         # Universal coding standards
│   │   ├── python.md          # Type hints, protocols, typing
│   │   ├── java.md            # Lombok, Optional, null-safety
│   │   ├── rust.md            # Ownership, Result types
│   │   ├── typescript.md      # Strict mode, readonly
│   │   ├── go.md              # Error handling, composition
│   │   └── c_cpp.md           # Smart pointers, RAII, const
│   └── context/               # Project memory (auto-populated)
│       └── README.md
└── .gitignore                 # Updated with .claude/context/

How it works

1

Run the installer

Execute the curl command in your project directory. The script auto-detects your languages.

2

Open in VS Code

Claude Code extension reads CLAUDE.md and .claude/rules/ automatically on startup.

3

Claude learns your project

On first interaction, Claude performs a deep dive and populates .claude/context/ with full project understanding.

4

Code with confidence

Every generated file follows all 10 principles, matches your existing patterns, and includes tests.
